About

This game is origanally made by Sparticle999, as a project to improve their skills at making games and to see what they could add to the incremental gaming world.
This desktop version is made by me, SpiderGamin. This project is just for fun and has helped me understand JavaScript.
The content in this game is not always accurate. This is because if it were accurate, many of the elements on other planets would be the same as Earth's, such as metal on Mars (iron oxide). Sparticle999 said, "Thus, I have had to make up some facts to help my game stay diverse. I apologise for any infuriation caused."
